MacPorts
www.macports.org

This installer installs MacPorts for Mac OS X 10.6 or later, up to macOS 10.13. There is a separate installer for older systems running Mac OS X 10.4 and 10.5.

You will be guided through the steps necessary to install this software. After MacPorts is installed, it will update itself.

Before installing MacPorts, please install Xcode. MacPorts uses Xcode compilers to update itself, and when installing some ports.
